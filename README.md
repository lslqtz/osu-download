# osu! 镜像下载客户端
## 主要特性
1. 支持选择路径。
2. 支持选择分支。
3. 支持增量更新。
4. 支持检测文件完整性。
5. 支持多 Mirror，自动检测延迟并进行升序排列。
6. 无需人工干预，可伪实时镜像最新 osu! 版本（附带服务端每 10 分钟检测一次新版本，上线服务端每 5 分钟检测一次新版本）。
